Create new 'Application'

A new application is the base unit for formloader, it contains all of the forms, fields, fieldsets, validations, and decorators that are associated with an application.

This is useful if you wish to have two similarly named fields which have different metadata and functionality (i.e. an Admin app and a Frontend app, or a Server Side & Client Side App)
